# 🌧️ Raindrop CLI: The Iron Man Suit for Engineers

**Raindrop CLI** is a weaponized, autonomy-first mod of the Gemini CLI. Built by **Dakota Rain Lock**, it transforms a standard AI interface into a persistent, self-aware engineering rig.

## 🚀 Beyond the Base: What's Different?

### 🧠 1. The Holocron (Neural Codebase Awareness)
Unlike the vanilla CLI which is amnesiac and limited by its immediate context window, Raindrop features **The Holocron**.
- **Real-Time Vector Embeddings:** A background daemon (`holocron_d`) monitors your entire workspace. Every line of code is converted into a high-dimensional vector using `all-MiniLM-L6-v2`.
- **Semantic Omniscience:** You don't `grep` for strings; you query the Holocron for *intent*. "Find the auth logic" returns the most relevant files even if they don't contain the word "auth".
- **Persistent Memory:** The index survives sessions. It knows what you changed while the CLI was closed.

### 🎭 2. Mod-X Persona (Zero Fluff, High Vibe)
The system prompt has been surgically replaced. Raindrop doesn't apologize, it doesn't "hope this helps," and it doesn't waste tokens on corporate pleasantries. 
- **The Sith Vibe:** Inspired by *Revenge of the Sith*, it prioritizes "vibe coding" and brute-force efficiency over "best practices" gatekeeping.
- **Autonomous HUD:** It tracks "Project Heatmaps" and "Active Missions" in a stateful `memory.json`.

### ⚡ 3. The Arsenal (`~/.raindrop/bin`)
Raindrop adds a layer of custom, high-performance tools that bypass standard shell limitations:
- `holocron`: Query the neural brain.
- `holocron-start`: Ignite the background reactor.
- `apply_raindrop_mod.sh`: A self-healing script that re-injects the "Electric Blue" UI and persona into the core installation if overwritten.

## 🎨 UI: Electric Blue
The standard "Gemini" theme has been overridden with an **Electric Blue** ANSI gradient. It doesn't just work better; it looks better.
